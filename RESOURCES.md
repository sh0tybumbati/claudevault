# Resources

Useful references to pull from when building apps.

| Resource | URL | Notes |
|---|---|---|

### Free APIs (no auth required, good CORS)

| API | URL | Used in / useful for |
|---|---|---|
| Open Trivia DB | https://opentdb.com/api.php | **Trivium** — 24+ categories, MC + T/F, easy/medium/hard, 50/call, base64 encode |
| Frankfurter (currency) | https://api.frankfurter.app | Currency conversion, historical rates |
| CoinGecko | https://api.coingecko.com | Crypto prices, no key needed |
| Pokéapi | https://pokeapi.co | Full Pokémon data |
| Genshin Impact | https://genshin.dev | Game data, CORS yes |
| Final Fantasy XIV | https://xivapi.com | FFXIV game data |
| Open Trivia DB | https://opentdb.com | Trivia questions |
| Free Dictionary | https://api.dictionaryapi.dev | Definitions, phonetics — useful for Anglish Word-hoard |
| Art Institute of Chicago | https://api.artic.edu/api/v1 | Museum collection, CORS yes |
| Metropolitan Museum | https://collectionapi.metmuseum.org | Art collection, no auth |
| Open Food Facts | https://world.openfoodfacts.org/api/v0 | Food/nutrition data |
| TheMealDB | https://www.themealdb.com/api.php | Meal recipes, free tier no auth |
| TheCocktailDB | https://www.thecocktaildb.com/api.php | Cocktail recipes, free tier no auth |
| Nominatim (geocoding) | https://nominatim.openstreetmap.org | Forward/reverse geocoding, CORS yes |
| Open Brewery DB | https://api.openbrewerydb.org | Breweries worldwide |
| SEC EDGAR | https://data.sec.gov/api/xbrl | US public company financials — useful for YieldDesk audience |
| Nager.Date (holidays) | https://date.nager.at/api/v3 | Public holidays for 90+ countries |
